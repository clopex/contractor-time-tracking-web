# Contractor Time Tracking SaaS Web

Standalone marketing site for the Contractor Time Tracking SaaS product.

## Pages

- Home
- Features
- Pricing
- Contact
- Privacy
- Terms

## Local development

1. Copy `.env.example` to `.env.local`.
2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```
